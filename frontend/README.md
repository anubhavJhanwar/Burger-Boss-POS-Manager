# Frontend

React + Vite frontend for Cafe POS system

## Setup

```bash
npm install
```

## Configuration

Create `.env` file:
```
VITE_API_URL=http://localhost:5000/api
```

## Run

```bash
npm run dev
```

App runs on `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Modern glassmorphism UI
- Responsive design
- Real-time cart updates
- Excel menu upload
- Dashboard analytics with charts
- Inventory management
- Expense tracking
